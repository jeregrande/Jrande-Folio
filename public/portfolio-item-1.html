Pulseye IoT Soil Monitoring

 August - December, 2019 

Github Repo

An IoT soil-monitoring system consisting of smart sensors connected to a web application that serves as the user interface to monitor soil conditions. Designed and implemented a responsive web interface using HTML, CSS, JS, related JS libraries, and Firebase services as the sole developer. This project fulfilled SJSU's engineering capstone project requirement.

While commercial options are available for smarter irrigation monitoring, there are no readily-available and convenient tools that can provide the same utility for residential use at affordable prices.

Pulseye offers a simple solution: a smart soil sensor device that captures specific soil information and provides it to its users through a simple web application interface. The system then tracks the soil parameters which allow for analysis of optimal watering-schedules via statistical modeling. Pulseye offers a better way for the common consumer to maintain efficient irrigation practices.
